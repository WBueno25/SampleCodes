# SampleCodes

Este repositório possui Alguns códigos que desenvolvi ao longo do tempo para exposição.

Na pasta PythonCodes há um desafio de programação resolvido por mim na linguagem python, o problema está proposto no readme da pasta e seu sua resolução e desenvolvimento estão explicados em um relatório. O código está comentado todos os arquivos necessários para execução estão na pasta em questão.

Na pasta System Verilog estçao alguns módulos de um processador RISC-V assíncrono escritos por mim, estes módulos são apenas exemplos e não estão todos inclusos pelo motivo de que o projeto completo ainda não irá ser disponibilizado publicamente até que se tenha feito uma publicação de paper em conferência ou revista, até lá apenas módulos exemplo estão sendo mostrados.
